# 0x00. Linux Security Basics

This directory contains bash scripts for fundamental Linux security tasks, including network monitoring, firewall configuration, and system auditing.

## Tasks
* **0-login.sh:** Shows the last 5 login sessions.
* **1-active-connections.sh:** Displays active network connections.
* **2-incoming_connections.sh:** Allows incoming traffic on port 80/tcp using UFW.
* **3-firewall_rules.sh:** Lists firewall rules using iptables.
* **4-network_services.sh:** Shows active network services and ports.
* **5-audit_system.sh:** Runs a system audit using Lynis.
* **6-capture_analyze.sh:** Captures and analyzes network packets using tcpdump.
* **7-scan.sh:** Performs a network scan using nmap.
