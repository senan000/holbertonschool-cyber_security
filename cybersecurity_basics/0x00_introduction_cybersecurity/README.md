# Cybersecurity Basics

This project introduces basic cybersecurity concepts using Bash scripting.

## Tasks:
- Display system distributor ID
- Generate strong random passwords
- Verify file integrity using SHA256
- Generate SSH key pairs
- Monitor user processes

All scripts are written in Bash and follow strict constraints (minimal lines, no unnecessary tools).

## Concepts covered:
- /dev/urandom
- SHA256 hashing
- SSH key generation
- Process monitoring
- Linux commands (ps, grep, tr)
